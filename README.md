# Actividad 2.6 - Cheerio

## Objetivo
Procesar HTML en backend usando Node.js, Express y Cheerio.

## Tecnologías
- Node.js
- Express
- Cheerio

## Instalación

```bash
npm install
```

## Ejecución

```bash
node server.js
```

## Endpoint

GET /scrape

## Datos extraídos
- Nombre
- Precio
- Categoría

## Selectores utilizados
- .producto
- .nombre
- .precio
- .categoria
